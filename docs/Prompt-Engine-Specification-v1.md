# KHODIM Prompt Engine Specification v1.0

## Vision
Prompt Engine adalah inti KHODIM Design Studio.

## Pipeline
User Input
→ Validation
→ Content
→ Brand
→ Layout
→ Visual
→ Photography
→ Islamic Rules
→ Negative Prompt
→ Prompt Optimizer
→ Prompt Specification JSON

## Content Types
- Promosi
- Dakwah
- Agenda
- Informasi
- Prestasi
- Hari Besar

## Output
Provider-independent Prompt Specification JSON.
